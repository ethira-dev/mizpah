# mizpah site

Astro + Tailwind marketing site and docs, deployed to GitHub Pages at
[ethira-dev.github.io/mizpah](https://ethira-dev.github.io/mizpah/).

```bash
just site-dev     # http://localhost:4321/mizpah/
just site-build   # output -> site/dist
```

Chrome / Cursor / Claude marks in the flow diagram come from
[Simple Icons](https://simpleicons.org/) (bundled, no API key). Landing-page
Claude / Codex / Grok consoles use [brainless](https://brainless.swerdlow.dev/)
session components.
